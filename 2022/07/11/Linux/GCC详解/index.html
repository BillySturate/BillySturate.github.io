GCC详解

Created2022-07-11|Updated2023-01-28|Linux
|Word count:1.6k|Reading time:5min|Post View:

视频教程

前言：

  1. GCC 编译器支持编译 Go、Objective-C，Objective-C ++，Fortran，Ada，D 和 BRIG（HSAIL）等程序；
  2. Linux 开发C/C++ 一定要熟悉 GCC
  3. VSCode是通过调用GCC编译器来实现C/C++的编译工作的；

实际使用中：

  • 使用 gcc 指令编译 C 代码
  • 使用 g++指令编译 C++ 代码

1. 编译过程

  1. 预处理-Pre-Processing //.i文件
  1. 编译-Compiling // .s文件
  1. 汇编-Assembling // .o文件
  1. 链接-Linking // bin文件

2. g++重要编译参数

  1. -g 编译带调试信息的可执行文件
  1. -O[n] 优化源代码
  1. -l 和 -L 指定库文件 | 指定库文件路径
  1. -I 指定头文件搜索目录
  1. -Wall 打印警告信息
  1. -w 关闭警告信息
  1. -std=c++11 设置编译标准
  1. -o 指定输出文件名
  1. -D 定义宏
